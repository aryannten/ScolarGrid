ScholarGrid Backend Setup

1. Install Docker Desktop.
2. Clone this repository.
3. Copy `.env.example` to `.env`.
4. Fill in the required values in `.env`.
5. Put `firebase-credentials.json` in the project root.
6. Run:

docker compose up --build

Backend URL:
http://localhost:8000

Notes:
- PostgreSQL will run on localhost:5432
- Redis will run on localhost:6379
- The `migrate` container exits after running migrations. That is expected.
